# Megacore International Website

High-impact, futuristic B2B sportswear manufacturing website built with:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui style components

## Run

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
